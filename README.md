# collabor8-video-calling
Collabor8 is an interactive video-calling platform where users can create their accounts and engage in live brainstorming sessions with other users. They can schedule meetings with ease, and have access to previous meeting history and their recordings if available.

### Features:
+ Seamless Signin/Signup with [Clerk](https://clerk.com/) Authentication
+ Video calling service by [getStream.io]([Link text Here](https://link-url-here.org))
+ Easy Scheduling for upcoming meetings
+ Access to Previous Meeting History
+ Storage for Meeting Recordings (If created)

### Future Updates:
+ AI Assistant Support during Live Meeting
+ Collaborative Whiteboard

### Tech Stack Used:
+ Next.js
+ shadcn ui/tailwind css

### Installation
    docker run -d -p 3000:3000 maniksharma17/collabor8
